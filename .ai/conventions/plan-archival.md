# Plan Archival

When a plan's implementation is fully complete (all checkboxes checked, branch merged or PR created), move the plan and its corresponding spec to their `done/` subdirectories:

- `docs/superpowers/plans/<name>.md` → `docs/superpowers/plans/done/<name>.md`
- `docs/superpowers/specs/<name>.md` → `docs/superpowers/specs/done/<name>.md`

Do this as part of the final commit when finishing a development branch.
